# Smart Classroom Understanding & Engagement Analyzer

## Problem
Teachers lack real-time insight into student engagement and understanding in online or hybrid classrooms.

## Solution
A privacy-first, AI-assisted dashboard that analyzes classroom interaction signals and provides actionable insights to improve teaching effectiveness.

## Features
- Real-time engagement visualization
- Confusion detection timeline
- Ethical, non-identifying analytics
- Teacher-friendly interface

## Tech Stack
- Frontend: React, Chart.js
- Backend: Node.js, Express
- AI Logic: Explainable rule-based analysis

## How to Run Locally
### Backend
```bash
cd backend
node server.js
cd frontend
npm start
